Jeśli nie widzisz: kliknij Releases w menu „About” po prawej, lub dopisz w adresie /releases/new. Utwórz Release Tag version: wpisz np. sky-runner-v1.0.0 i kliknij Create new tag on publish. Release title: np. Sky Runner v1.0.0. Przewiń do sekcji Assets i przeciągnij plik .apk (np. sky-runner-v1.0.0.apk). Kliknij Publish release. Każda gra = osobny Release z unikalnym tagiem i jednym plikiem .apk. B) Podłącz grę na stronie (repo „portfolio”) Otwórz plik strony Wejdź do repo portfolio → kliknij plik index.html.
